# Poly-Lang Chrome Extension

Webページ上のテキストを選択して翻訳・フレーズ保存ができるChrome拡張機能です。

## 機能

- **クイック翻訳**: Webページ上でテキストを選択すると翻訳ボタンが表示されます
- **右クリックメニュー**: 選択したテキストを右クリックして「Poly-Langで翻訳」を選択
- **フレーズ保存**: 翻訳したフレーズをPoly-Langアカウントに保存
- **ポップアップ翻訳**: 拡張機能アイコンをクリックして直接テキストを入力・翻訳
- **設定同期**: Poly-Langアプリと言語設定が同期されます

## インストール方法

### 開発者モードでインストール

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `chrome-extension` フォルダを選択

### アイコンの準備

現在SVGアイコンのみが含まれています。PNGアイコンを生成するには:

1. `icons/icon.svg` を開く
2. 以下のサイズでPNGとしてエクスポート:
   - `icon16.png` (16x16)
   - `icon48.png` (48x48)
   - `icon128.png` (128x128)

または、以下のオンラインツールを使用:
- https://svgtopng.com/

## 使い方

### 1. ログイン

1. 拡張機能アイコンをクリック
2. Poly-Langアカウントのメールアドレスとパスワードを入力
3. ログインボタンをクリック

### 2. テキストを翻訳

**方法A: テキスト選択**
1. Webページ上で翻訳したいテキストを選択
2. 表示される翻訳ボタンをクリック
3. 翻訳結果がポップアップで表示されます

**方法B: 右クリックメニュー**
1. テキストを選択
2. 右クリック → 「Poly-Langで翻訳」を選択

**方法C: ポップアップで直接入力**
1. 拡張機能アイコンをクリック
2. テキストを入力
3. 翻訳ボタンをクリック

### 3. フレーズを保存

1. 翻訳結果が表示されたら「保存」ボタンをクリック
2. フレーズがPoly-Langアカウントの「Chrome Extension」セットに保存されます
3. Poly-Langアプリの「フレーズ」ページで確認できます

## 設定

拡張機能アイコン → 設定ボタン（歯車アイコン）から以下の設定を変更できます:

- **母国語**: 翻訳元の言語
- **学習中の言語**: 翻訳先の言語
- **選択時に自動翻訳**: テキスト選択時に自動で翻訳を開始
- **翻訳ポップアップを表示**: テキスト選択時にポップアップを表示

## 技術仕様

- Manifest V3
- Chrome Storage API (認証情報の保存)
- Content Scripts (ページ内のテキスト選択検出)
- Service Worker (バックグラウンド処理)
- Supabase (認証・データベース)

## トラブルシューティング

### ログインできない
- Poly-Langアカウントのメールアドレスが認証済みか確認してください
- パスワードが正しいか確認してください

### 翻訳が表示されない
- インターネット接続を確認してください
- ページをリロードしてみてください

### フレーズが保存されない
- ログインしているか確認してください
- Poly-Langアプリで「Chrome Extension」フレーズセットを確認してください
